# Learning Process  

## Problem 1
Feynman Technique is method in which we try to explain a particular topic in simpe words, as if teaching it to a child.

## Problem 2
A great story from talk is the anecdote about how they would relax in a chair holding an object and as they drifted off to sleep, the object would drop, waking them up so they could capture creative connections their resting brains.

## Problem 3
* **Active Mode:** Concentrated state brain enters when trying to learn something specific or solve a problem familiar with.
* **Diffused Mode:** A relaxed, resting state of the brain. It allows the brain to subconsciously connect  ideas and solve less familiar problems.

## Problem 4

1. Break the skill into smaller, manageable skills.
2. Learn just enough to correct during self practice.
3. Remove physical, mental, and emotional barriers to practice.
4. Practice the most important skills for at least 20 hours.

## Problem 5

* Commit to 100% involvement by silencing phone, turning off non-essential notifications, and using site-blockers during work hours.
* When learning software concepts, ask self what the concept is, where it is applied, and how to implement it in code.
* Explain concepts in own words.
* To speak up clearly and precisely about any obstacles or difficulties faced.
